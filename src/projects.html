Hvid zone 2013

As a part of my job at an agency, developed to promote the educations for radiology, bio-analytics and nursing, This is a real-time quiz app meant to work on both desktop and mobile browsers with a twist of Facebook.

The app leads the user through diagnosing and choosing a treatment of 40 Facebook friends in a given medical field. Based on how well the user does a score is given and if the score is high enough the user is recommended to apply for the given education.

This was made as web app backend when web applications was not really a thing yet, i set up full MVC and templating using Mustache, Backbone and underscore.
